# 🛒 Local Mart - UI/UX Design
UI/UX design for Local Mart, a mobile grocery shopping application created in Figma. Includes home screen, login, registration, product listings, and user interface prototypes.
A modern mobile grocery shopping application designed in **Figma**. This project demonstrates the complete user interface design for an online grocery shopping experience, from user authentication to product browsing.

## 📱 Screens Included

- 🏠 Home Page
- 🔐 Login Page
- 📝 Registration Page
- 📂 Menu Screen
- 🛍️ Product Page
- 🖼️ Complete Design Layout
